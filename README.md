# LangGraph for Beginners

## 📌 Project Overview
This project is a beginner-friendly implementation of LangGraph, covering the fundamentals of building graph-based workflows and integrating Large Language Models (LLMs). It is divided into two parts: creating a simple graph structure and extending it to call an LLM for intelligent responses. The goal is to help developers understand how stateful AI workflows are built using graph-based execution.

---
## 🚀 Features
- Build a simple LangGraph workflow from scratch
- Understand nodes, edges, and state management
- Integrate LLM calls into the graph
- Step-by-step implementation for beginners
- Clean and minimal code structure for learning

---
## 🧠 Technologies Used
- Python
- LangGraph
- LangChain
- OpenAI API
- dotenv

---
## ⚙️ How It Works
1. Define the state structure that flows through the graph
2. Create nodes (functions) that process the state
3. Connect nodes using edges to form a graph
4. Compile the graph into an executable workflow
5. Extend the graph to call an LLM for generating responses
6. Execute the graph with input and observe outputs
